# Theme customization

The custom theme entry extends the default VitePress theme and imports:

- the shared stylesheet from `src/vitepress/styles.css`,
- `uno.css` so utility classes remain available.

This keeps the playground close to a real consumer project while still proving that the shared design tokens and layout tweaks from the library render correctly.

If you want to go further, this is the place to add custom layout slots, components, or extra CSS tokens.
