# VitePress config

`defineDocsVitepressConfig(...)` wraps the base VitePress configuration and injects shared defaults for:

- metadata and social tags,
- sitemap settings,
- local search,
- footer and social links,
- the `vitepress-plugin-llms` integration.

The playground keeps the custom part focused on navigation and sidebar structure so the example stays easy to read.

You can use this page to verify that nested config resolution works correctly from `docs-playground/.vitepress/config.mts`.
